== Description ==
Adsense is a free plugin and open source software that automatically insert Google Adsense ads into Ushahidi frontend pages.

To make money with Google Adsense ads you just have to have an account with Google Adsense.

You have full control over the ads. It is possible to control color, size, number and types of ads. You determine how many Adsense ads you want to show.


== Installation ==
Copy the entire /adsense/ directory into your /plugins/ directory.
Activate the plugin.
